== Description ==

This is a WordPress extension that extends WooCommerce. It includes the necessary JavaScript and PHP WordPress plugin. The template for creating the widget can be found on the plugin's setting page. This widget requires both WooCommerce and YITH Affiliates (for now). The tweaks are designed specifically for https://MarkPorterLive.com but can be applied anywhere. Credit is given for sources of inspiration.

== Installation ==

1. Upload the entire 'plugin' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
